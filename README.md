เครื่องมือวัดและแปลผลการบรรลุ PLO (PLO Assessment Tool)

เครื่องมือกรอกคะแนน rubric รายคน แล้วคำนวณและ แปลผลการบรรลุผลลัพธ์การเรียนรู้ระดับหลักสูตร (PLOs) ของนักศึกษาทั้งรุ่นโดยอัตโนมัติ
สำหรับหลักสูตรศิลปศาสตรบัณฑิต สาขาวิชาสื่อสารการกีฬา (หลักสูตรปรับปรุง พ.ศ. 2568)
คณะศิลปศาสตร์ มหาวิทยาลัยการกีฬาแห่งชาติ วิทยาเขตกรุงเทพ

เป็นไฟล์ HTML ไฟล์เดียวจบ (plo-assessment-tool.html) ไม่ต้องลงโปรแกรม ไม่ต้องมีเซิร์ฟเวอร์
เปิดด้วยเบราว์เซอร์ได้ทันที หรือนำขึ้น GitHub Pages เพื่อแชร์ลิงก์

วิธีใช้ 3 ขั้น


แท็บ รายชื่อ — เพิ่มนักศึกษา (พิมพ์ทีละคน หรือวางทั้งรายชื่อแบบ รหัส เว้นวรรค ชื่อ) หรือกด “ใส่ตัวอย่าง 6 คน” เพื่อลองดูก่อน
แท็บ กรอกคะแนน — เลือก PLO ที่จะกรอก แล้วให้คะแนนแต่ละเกณฑ์ rubric เป็นระดับ 1–4 (เว้นว่างได้ถ้ายังไม่ประเมิน) ช่อง “เฉลี่ย” และ “ผล” ขึ้นทันที
แท็บ ผลวิเคราะห์ — เห็น % การบรรลุแต่ละ PLO พร้อมคำแปลผล เทียบเป้าหมาย และรายชื่อที่ควรติดตาม


ตรรกะการแปลผล


คะแนนเฉลี่ยของ PLO ของนักศึกษา ≥ เกณฑ์ผ่าน (ตั้งต้น 3 จาก 4) = บรรลุ PLO นั้น
ต้องกรอกครบทุกเกณฑ์ของ PLO นั้นจึงนับว่า “ประเมินแล้ว”
% บรรลุของรุ่น = จำนวนคนที่ผ่าน ÷ จำนวนคนที่ประเมินครบ
เทียบกับ เป้าหมายของรุ่น (ตั้งต้น 70%) แล้วติดป้าย: บรรลุเป้าหมาย / ใกล้เป้าหมาย / ต่ำกว่าเป้าหมาย
ระบุระดับทักษะของรุ่น: ดีมาก (≥3.5) · ดี (≥3) · กำลังพัฒนา (≥2) · เริ่มต้น (<2)


ปรับให้ตรงหลักสูตร (แท็บ ตั้งค่า)


แก้ ชื่อจุดวัด (วิชา/ชิ้นงาน) และ เกณฑ์ย่อย rubric ของแต่ละ PLO
ปรับ มาตรวัด (เต็ม 4 หรือ 5), เกณฑ์ผ่าน, และ เป้าหมาย % ของรุ่น


ค่าตั้งต้นของจุดวัด (อิงคู่มือการวัดและประเมินผล):


PLO 1 (ความรู้) → สอบประมวลความรู้ / วิชาแกนทฤษฎี (ปี 4)
PLO 2 (ทักษะ–ภาษาอังกฤษ) → TNSU English Test / ภาษาอังกฤษวิชาชีพ
PLO 3 (ทักษะ–ผลิตสื่อ) → โครงงาน/ชิ้นงานผลิตสื่อ (ปี 3–4)
PLO 4 (จริยธรรม/ลักษณะบุคคล) → รู้เท่าทันสื่อ + ประเมินพี่เลี้ยงฝึกฯ (ปี 4)


ข้อมูลและการส่งออก


บันทึกอัตโนมัติในเบราว์เซอร์เครื่องนี้ (แท็บเดียว/เครื่องเดียว)
บันทึกไฟล์ .json เพื่อสำรอง/ย้ายเครื่อง และ โหลดไฟล์ .json กลับมาได้
ส่งออก CSV ไปเปิดใน Excel · พิมพ์เป็น PDF เพื่อแนบเป็นหลักฐาน AUN-QA



ข้อมูลนักศึกษาเก็บอยู่ในเครื่องที่กรอกเท่านั้น (ไม่ได้ส่งขึ้นเซิร์ฟเวอร์) หากใช้หลายเครื่อง ให้ใช้ไฟล์ .json โอนข้อมูลกัน
